# Floor Slope Analysis ‚Äì Revit Add-in

A Revit add-in for analyzing and visualizing floor slopes directly from geometry.  
Designed for BIM workflows where accurate slope detection and quick visual feedback are essential.

---

## üîç Overview

**Floor Slope Analysis** allows users to select multiple floor elements and automatically evaluate their slopes based on actual geometric data.  
The add-in works with flat floors, sloped floors, and slab-shaped floors without modifying any geometry.

---

## ‚ú® Features

- Manual selection of multiple floor elements
- Supports:
  - Flat floors
  - Sloped floors
  - Slab-shaped floors (shape-edited floors)
- Calculates slope percentage from geometry
- Detects top planar faces automatically
- Visual slope feedback using color indicators
- Safe cancel handling during selection
- No permanent model changes

## üß™ How to Use

1. Open a Revit project.
2. Click **Floor Slope Analysis** from the ribbon.
3. Select one or more floor elements.
4. Slopes are analyzed and visualized directly in the model.
